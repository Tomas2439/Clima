# Clima 🌦️

¡Bienvenido a mi app del Clima! Una aplicación meteorológica basica y elegante que te trae la información del tiempo más actualizada.

Utilizando la API de [OpenWeatherMap](https://openweathermap.org/api), la interfaz de la aplicación cambia dinámicamente para reflejar el clima actual en la ubicación que elijas. ¡Si está lloviendo afuera, también lo estará en tu pantalla!

## ✨ Características Principales

- **Clima en Tiempo Real:** Obtén datos meteorológicos actualizados al instante.
- **Búsqueda por Ciudad:** Consulta el clima de cualquier ciudad del mundo.
- **Geolocalización:** Detecta automáticamente tu ubicación para mostrar el clima local.
- **Interfaz Dinámica:** El fondo y los íconos se adaptan a las condiciones climáticas (soleado, nublado, lluvia, nieve, etc.).
- **Información Detallada:** Muestra la temperatura, sensación térmica, humedad y velocidad del viento.

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Para la estructura de la aplicación.
- **CSS3:** Para los estilos y el diseño visual.
- **JavaScript (Vanilla):** Para la lógica de la aplicación y la comunicación con la API.
- **API:** [OpenWeatherMap API](https://openweathermap.org/api) para los datos meteorológicos.

## 🚀 Cómo Empezar

Para poner en marcha este proyecto en tu máquina local, sigue estos pasos:

1.  **Clona el repositorio:**
    ```bash
    git clone https://github.com/tu-usuario/Clima.git
    cd Clima
    ```

2.  **Configura tu API Key:** 
    -   Este proyecto requiere una clave de API de OpenWeatherMap.
    -   Consigue una clave gratuita registrándote en OpenWeatherMap.
    -   Dentro de tu archivo JavaScript (ej. `script.js`), busca la variable donde se almacena la API key y reemplaza el valor de ejemplo por tu clave.

3.  **Abre la aplicación:**
    -   ¡No necesitas instalar nada! Simplemente abre el archivo `index.html` en tu navegador favorito.
